## Trust Boundaries

| Boundary | Description |
|---|---|
| verified facts -> plan | 计划不得引入、删改或隐匿事实来满足分页。 |
| plan -> Typst | 不可信/篡改计划不得取得发射能力。 |
| Typst flow -> page contract | 流式结果不能扩为第三页或截断条目。 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-47-11 | Tampering | FrozenResumePlan | mitigate | frozen dataclass、source_fact_ids 对照、plan.validate 和计划/主题策略相等性检查。 |
| T-47-12 | Spoofing | container projection | mitigate | 仅 validate 后事实可进入容器；空模块不渲染，禁止占位和生成内容。 |
| T-47-13 | Denial of Service | long CJK content | mitigate | 固定字号/安全区/预算；无合法 1/2 页即 `LAYOUT_UNSATISFIABLE`，不裁断或生成第三页。 |
| T-47-14 | Information Disclosure | plan JSON/Typst | mitigate | projection allowlist，no-photo 删除媒体字段，发射器只用验证后的句柄。 |

| T-47-SC | Tampering | emitter | mitigate | 机械消费冻结 pages，拒绝重排、二次路径读取、未许可裁切和 stretch。 |
