## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI args/Markdown → services | 用户可控 target IDs、overrides、condition reasons、路径和候选文本进入解析、渲染与发布。 |
| preflight output/confirmation → mutation | delta 与 gap 允许必须绑定本次未变化快照，不能复用旧确认。 |
| internal evidence → public CLI/root | condition reasons、事实正文、联系方式、机器路径与照片中间件不得泄露。 |
| render/batch orchestration → current/history | partial candidate、单 target failure 或 stale root 不得形成部分当前版本。 |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
