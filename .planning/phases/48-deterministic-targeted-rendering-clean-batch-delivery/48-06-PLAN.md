## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| condition evaluator → public/evidence | 公开只允许有界四态；完整理由和事实 ID 仅进入授权隐藏证据。 |
| evidence root → local filesystem | caller-controlled 路径可能是 symlink、投递根子树、unknown 或 stale 状态。 |
| history → later authority session | 旧 stem 不再属于 current authority，但仍是不可信的持久文件系统输入。 |
| preflight process → confirm process | current、canonical、candidate 或目标集合可在用户审阅后变化，形成 TOCTOU。 |

| T-48-G06-02 | Tampering | condition evidence | mitigate | canonical JSON bytes 绑定 canonical/projection/condition digests；保存后重读重算。 |
