## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Python snapshot fd → helper | 同 UID 攻击者可在 Python hash 后原地覆写 snapshot；fd 与 digest 必须由 helper 再绑定为 root-owned copy。 |

| helper → Typst argv/output | 用户输入必须保持既有受控 argv、输出大小与 timeout 边界；helper 不得以 root 身份运行 Typst。 |
