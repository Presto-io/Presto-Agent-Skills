<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| assets root → photo bytes | 本地目录和 leaf 可含 symlink、特殊文件、超限内容或验证后替换。 |
| PATH → Typst source descriptor | PATH 候选可为 Homebrew symlink，也可为错误版本、cycle、伪造工具或复制期间 identity swap。 |
| source descriptor → private snapshot | 若复制不绑定同一 descriptor/bytes/hash，version 与实际 compile 仍可能不是同一工具。 |
| verified runtime → three consumers | 任一消费者自行重新解析命令会绕过共同版本门禁。 |
| delivery context manager → CLI result | cleanup failure 可能在 current 已发布或已 rollback 后覆盖公开返回语义。 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-48-G08-01 | Tampering | photo asset | mitigate | held root + descriptor-relative `O_NOFOLLOW` + single bounded bytes snapshot。 |
| T-48-G08-02 | Spoofing | Typst source resolution | mitigate | bounded/cycle-safe symlink chain、final `O_NOFOLLOW` descriptor、snapshot `--version` exact 0.15.0；真实 Homebrew symlink 为正例。 |
| T-48-G08-03 | Tampering | executable substitution | mitigate | 从 held source descriptor 复制并 hash/fsync 到私有 `0700` 目录的 `0500` snapshot；version/fonts/photo/PDF 只执行同一 snapshot，原路径在 final check 与 exec 间替换无效。 |
| T-48-G08-04 | Information Disclosure | final Typst | mitigate | 只内嵌 normalized PNG bytes，不输出原始路径、EXIF 或 URL。 |
| T-48-G08-05 | Denial of Service | photo/tool output | mitigate | bounded file bytes、bounded version output、subprocess 非零统一失败。 |
| T-48-G08-06 | Repudiation | cleanup CLI result | mitigate | 公开 CLI cleanup fault 负例要求 bounded non-zero、无 success payload，并核对 published/rollback current。 |
| T-48-G08-SC | Tampering | package supply chain | accept | 不安装包；只验证仓库已锁定的本地 Typst 0.15.0。 |
</threat_model>
