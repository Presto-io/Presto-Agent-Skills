<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Markdown hierarchy -> renderer parser | Author-controlled headings and slide metadata become navigation structure and page records. |
| Renderer manifest -> generated HTML state | Manifest/page records drive DOM attributes, hash routing, preview selection, rail selection, and overview selection. |
| User media paths -> offline package | Existing image/video handling remains constrained to allowed relative media and skill-local identity assets. |
| Embedded JavaScript -> local browser behavior | Generated single-file HTML runs stateful navigation logic without a server or framework. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-01 | Tampering | hierarchy parser | mitigate | Preserve flat `## Slide:` compatibility, parse explicit `## Section:` conservatively, and verify default-section fallback. |
| T-09-02 | Spoofing | current-page URL hash | mitigate | Use renderer-owned page ids and bounds checking before selecting a page from hash. |
| T-09-03 | Tampering | DOM state sync | mitigate | Route rail, preview, overview, playback, and hash updates through one `selectPage` state transition. |
| T-09-04 | Denial of Service | thumbnail rendering | mitigate | Use CSS-scaled real DOM only where practical and layout-family cards in constrained surfaces; avoid screenshot generation. |
| T-09-05 | Information Disclosure | generated page metadata | accept | Page titles and section labels are already visible deck content; do not add hidden source paths beyond existing manifest behavior. |
| T-09-06 | Elevation of Privilege | script execution/network | mitigate | Keep verification local; do not add package installs, remote fetches, servers, or external runtime dependencies. |
| T-09-SC | Tampering | Phase 10 scope bleed | mitigate | Keep reveal steps as empty placeholders and document ordered reveals, emphasis animation, and answer masks as Phase 10 only. |
</threat_model>
