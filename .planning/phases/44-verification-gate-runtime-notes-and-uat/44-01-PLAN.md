## ASVS L1 / block_on=high

| Threat | Risk | Mitigation and verification |
|---|---|---|
| 路径越界 | 恶意 `--workdir`、`..` 或内部 artifact 名写到 caller root 外 | held/no-follow root、component validation、relative path assertion；symlink/path-escape 黑盒负例。 |
| Symlink/TOCTOU | 检查后替换 workdir/evidence target，覆盖任意文件 | descriptor-relative ownership、no-follow create/replace、identity recheck；race fault 必须 fail closed 并保持 old target/sentinel。 |
| 无界或泄漏 evidence | subprocess/user text、绝对 Python/temp 路径、traceback、secret 进入 JSON/Markdown | 8KiB public bound、字段/collection caps、path sanitizer、同一 in-memory projection 与 mutation tests。 |
| Stale PASS | failed rerun 后旧成功 evidence 仍可见 | current failed record 原子替换，run-specific failure summary；不得读取旧 evidence 作为本次 operand。 |
| 依赖误报 | 常量 `ready:true` 或把 optional Pandoc 变 required | 真实 import/version operands、fault injection、Pandoc `required:false` schema assertion。 |
| 资源耗尽 | 无界 child process/output 或 run directories | child timeout、capture byte cap、bounded registry/diagnostics、唯一有限 run tree。 |
</threat_model>
