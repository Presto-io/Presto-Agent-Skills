| 诊断泄漏 | hard-split/code 错误回显整段源内容 | diagnostic 仅 code、位置、logical/physical index 和 fix；输出长度测试。 |
| 算法 DoS | 全局分区、字号组合或 grapheme splitter 指数增长 | contiguous DP 多项式上界、候选字号有限、超预算 fail bounded；压力测试 2048 blocks。 |
| Unicode 分割破坏 | 组合字符或 emoji 被切断导致内容篡改 | grapheme-safe iterator 与 round-trip 测试；源 code 行永不修改。 |
| PPTX/ZIP/XML/path/TOCTOU | 本计划不创建包或发布文件 | 明确不接触 PPTX、ZIP、输出路径与 replace；相关 High 风险由 Plan 03/04 阻断，不得提前引入 I/O。 |
