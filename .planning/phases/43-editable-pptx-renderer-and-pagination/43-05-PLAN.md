|---|---|---|
| staged symlink TOCTOU | 路径 save 先覆盖外部文件，事后才发现 inode 变化 | owner fd 持有到 replace；save/validate 使用 descriptor stream；精确 reserve/save 窗口交换并断言 sentinel byte-identical。 |

| 对象异常泄漏 | traceback、绝对媒体路径和库内部信息进入 CLI | code allowlist + 通用消息；黑盒 stdout/stderr 长度和禁词断言。 |
| 破坏性清理 | 失败清理跟随 symlink 或删除 caller 文件 | 仅在 held root fd 下核对同 inode 普通目录项后 unlink；不同 inode/symlink 允许保留，held renderer inode 由最后一个 fd 关闭回收。 |
