# Phase 05 Plan: Gongwen Skill

**Milestone:** v1.2 Gongwen Skill
**Created:** 2026-05-31
**Mode:** mvp

## Goal

Create the first concrete portable skill in this repository: `gongwen`, a markdown-first workflow for Chinese official-document style writing and rendering through the Presto gongwen Typst template.

## Scope

- Add `skills/gongwen/SKILL.md` as the canonical skill source.
- Add a reusable `gongwen-full.md` Markdown template that demonstrates the supported intermediate structure.
- Add script code that calls the Presto `presto-template-gongwen` binary, writes Typst output, compiles PDF with Typst, and can compare against reference Typst/PDF fixtures.
- Validate against the provided `gongwen-full.md`, `gongwen-full.typ`, and `gongwen-full.pdf` fixture set.

## Tasks

- [x] Define the skill contract, trigger wording, inputs, outputs, verification, and safety boundaries.
- [x] Implement scriptable Markdown-to-Typst/PDF rendering and fixture comparison.
- [x] Verify generated Typst and PDF against the provided fixture artifacts.

## Validation

The phase is complete when the script can generate Typst identical to the provided reference and compile a byte-identical PDF through Typst.
