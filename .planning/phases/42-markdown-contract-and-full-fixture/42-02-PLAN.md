| Boundary | Description |

| Canonical fixture/media -> copied example | Repository-owned bytes cross into a caller-controlled output tree and must not gain broader deletion/overwrite authority. |
| `--out-dir` -> filesystem | The caller may provide traversal paths, symlinks, directories at file locations, unwritable paths, or a directory containing unrelated valuable files. |

| Diagnostics -> terminal/evidence | Failure messages must explain collisions and invalid source without stack traces, file contents, or machine-sensitive source paths. |
