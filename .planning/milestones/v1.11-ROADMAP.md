## Overview

v1.11 created the integrated `teaching-design-package` orchestration path while preserving the existing `jiaoan-jihua`, `jiaoan-shicao`, and `end-of-term-teaching-materials` skills as standalone public workflows. The milestone promoted scheduling inference into an auditable package-level contract, added a persistent teacher-reviewable package Markdown checkpoint, connected jiaoan and optional end-of-term module handoffs, and verified runtime/documentation discoverability across the supported agent targets.

## Milestone Summary

**Decimal Phases:** None.

**Key Decisions:**

- Scheduling inference is documented as package-level behavior, with `jiaoan-jihua` serving as implementation evidence rather than a broad shared API.
- `teaching-design-package` composes the existing teaching skills rather than deleting, renaming, or replacing their standalone public entries.
- The package Markdown checkpoint is the persistent teacher-reviewable source before Typst/PDF/workbook generation.
- Optional end-of-term behavior delegates scoring, deterministic tables, workbook generation, PDF rendering, and module review resolution to `end-of-term-teaching-materials`.
- Combined package readiness requires actual `teaching-design-package.pdf` file evidence; split output success alone does not imply final readiness.

**Issues Resolved:**

- TDP-01 through TDP-16 are covered by phase artifacts and the passed milestone audit.
- Public command regression passed for all affected teaching skills.
- Documentation/index discoverability was corrected for the new package skill.
- Manifest status now records disabled, planned, not_run, passed, failed, missing_compiler, merge_unavailable, and blocked_review states honestly.

**Issues Deferred:**

- Global shared scheduling library for unrelated skills.
- Hosted collaboration, cloud review, or online teacher approval workflow.
- Automatic inference for arbitrary school templates, OCR handwritten schedules, or uncontrolled spreadsheets.
- Physical deletion, renaming, or merging of original teaching skill directories.

**Technical Debt Incurred:**

- Combined PDF generation remains status-gated; future work can add real merge/compile tooling while preserving manifest truthfulness.
- Phase directories remain in `.planning/phases/` as raw execution history and can be archived later with cleanup tooling if desired.
