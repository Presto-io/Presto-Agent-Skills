# Cover Template Notes

这些封面说明是技能本地固定模板资源，使用脱敏 fixture 验证。真实学校模板、桌面文件、私有路径、教师签字图片或学生原始成绩不得写入本目录。

## 成绩记分册

`成绩记分册` 包含重新设计的封面和记分册正文。封面按学校参考样张使用大号居中标题、充分留白和三行下划线信息栏，信息栏只展示科目、班级和教师；正文沿用 Markdown 中的学生顺序、任务顺序和原始成绩单元格。

## 成绩分析册

`成绩分析表` artifact 包含成绩分析册封面和成绩分析表正文。成绩分析册封面与其他封面保持同一标题和留白节奏，信息栏采用一行合并科目、两列专业/班级、两列教师/日期和一行合并学期的下划线布局。

## 教学日志封面

教学日志封面使用参考样张的三行下划线信息栏，展示科目、班级和教师。没有来源字段时不猜测历史周次或日志页码。

## 过程考核评价表封面

过程考核评价表封面使用参考样张的三行下划线信息栏，展示科目、班级和教师。任务名称和课时仍来自 `## 过程考核任务`，但不在封面上展开。

## 交接班记录封面

交接班记录封面需要 `handover_class_name` 和 `handover_teachers`。两个字段任意一个缺失时，即使 package flags 请求 `交接班记录封面`，renderer 也必须跳过该封面并记录 manifest warning。

## Review Boundary

任何封面都不得绕过 Markdown 复核规则。只要 `## 复核标记` 不是精确的 `无`，最终 PDF/workbook export readiness 必须失败。
